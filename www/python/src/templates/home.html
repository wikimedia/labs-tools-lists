Welcome to Lists Project homepage!

Project's purpose

The goal of this project is to offer a list of lists, autoupdate every day or week (according to the importance), for Wikimedia projects. Each list is generated from the shared database, using the query written at the bottom of its page.

Currently the tool produces lists only for Italian Wikipedia. Write us if you want some lists implemented also in <any_language> Wikipedia!

Contacts

If you want new lists or if you have any questions regarding Lists, please contact its authors in this talk page on Italian Wikipedia (please write only in English or Italian).
